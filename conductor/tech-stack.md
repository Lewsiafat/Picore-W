# Tech Stack - Picore-W

## Hardware
- **MCU**: Raspberry Pi Pico 2 W (RP2350)
- **Networking**: CYW43439 (onboard WiFi/Bluetooth)

## Software & Runtime
- **Language**: MicroPython (optimized for RP2350)
- **Async Framework**: `uasyncio` (built-in)
- **Networking APIs**: `network`, `usocket` (async compatible)

## Development Tools
- **Deployment**: MicroPico (VS Code Extension) or `mpremote`
